# Semana 02 — Variables y Tipos de Datos

## 📌 Descripción
En esta semana se creó una ficha de datos del dominio utilizando variables en JavaScript.

Se aplicaron conceptos fundamentales como:

- Variables constantes
- Tipos de datos
- Template literals
- Verificación de tipos con `typeof`
- Conversión explícita de datos

## 🏠 Dominio del proyecto
**App de Valuación de Inmuebles**

El sistema representa información de propiedades inmobiliarias registradas dentro de una aplicación de avalúos.

## 📊 Datos manejados

Ejemplos de variables utilizadas:

- Nombre del inmueble
- Categoría de la propiedad
- Valor estimado del inmueble
- Estado de disponibilidad
- Avalúo pendiente (`null`)

## 🧠 Conceptos aplicados

Tipos de datos utilizados:

- `string`
- `number`
- `boolean`
- `null`
