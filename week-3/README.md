# Semana 03 — Operadores y Cálculos

## 📌 Descripción
En esta actividad se construyó una calculadora básica aplicada al dominio asignado.

Se realizaron diferentes cálculos relacionados con el valor de propiedades inmobiliarias.

## 🏠 Dominio del proyecto
**App de Valuación de Inmuebles**

El sistema permite calcular valores importantes dentro del proceso de avalúo inmobiliario.

## 📊 Cálculos implementados

- Comisión de venta inmobiliaria
- Valorización estimada del inmueble
- Promedio de valores evaluados
- Proyección del valor del inmueble a futuro
