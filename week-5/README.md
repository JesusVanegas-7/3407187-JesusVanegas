Semana 05 — Clasificador

Se implementa un sistema de clasificación de inmuebles usando:

if / else → Clasificación por precio
Operador ternario → Estado (disponible/no disponible)
switch → Tipo de inmueble
?? → Valores por defecto
?. → Acceso seguro a propiedades

📌 Resultado: Se genera una ficha de clasificación del inmueble.
