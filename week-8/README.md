Semana 08 — Gestión de Inventario

Se implementa un sistema completo de gestión:

📦 Operaciones CRUD básicas:
push → Agregar inmueble
pop → Eliminar último
unshift → Agregar prioritario
splice → Eliminar por índice
🔎 Métodos avanzados:
filter → Inmuebles disponibles
find → Búsqueda por nombre
map → Transformación de datos
spread (...) → Copia inmutable

📌 Resultado: Simulación de un inventario dinámico tipo backend.
