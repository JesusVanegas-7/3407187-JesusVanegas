# Semana 04 — Generador de Mensajes

## 📌 Descripción
En esta actividad se desarrolló un generador de mensajes utilizando métodos de manipulación de strings en JavaScript.

El sistema genera una ficha informativa de un inmueble utilizando diferentes transformaciones de texto.

## 🏠 Dominio del proyecto
**App de Valuación de Inmuebles**

El sistema permite registrar información de propiedades inmobiliarias dentro de una plataforma de avalúo.

## 🧠 Métodos de string utilizados

Se aplicaron diferentes métodos de JavaScript:

- `trim()` → limpiar espacios
- `toUpperCase()` → convertir texto a mayúsculas
- `toLowerCase()` → convertir texto a minúsculas
- `slice()` → extraer partes de un string
- `startsWith()` → validar prefijo
- `includes()` → buscar palabras
- `endsWith()` → validar sufijo
- `repeat()` → generar separadores

## 📊 Funcionalidad

El programa:

1. Limpia y transforma datos del inmueble
2. Genera una ficha informativa en consola
3. Valida información del código y descripción
4. Muestra una notificación de registro del inmueble
